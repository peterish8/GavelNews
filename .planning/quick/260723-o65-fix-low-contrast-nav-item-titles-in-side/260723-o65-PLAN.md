# Quick Task 260723-o65: Fix low-contrast nav item titles in Sidebar

**Created:** 2026-07-23

## Problem

User feedback: "still nav bar text inside that is too blend with the background
so not visible much want black text for titles in the nav bar!"

In `components/Sidebar.tsx`, `SidebarLink`'s inactive-state `<Link>` sets
`text-ink-2` (medium purple-gray, `#434056`) on the whole row, and the label
span (`item.label` — "Today", "Calendar", "Search", etc.) had no explicit
color override, so it inherited that muted tone instead of the near-black
`--ink` (`#130F2A`) used for the wordmark and other real titles.
